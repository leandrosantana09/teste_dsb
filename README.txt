# teste_dsb


O seguinte repositorio contem uma proposta simples para solução do problema proposto pela vaga de 
cientista de dados da Data Science Brigade, desenvolvida por Leandro Santana, candidato.

Linkedin: https://www.linkedin.com/in/santana-leandro/

# Pontos importantes:

-Todos os dados ultilizados foram retirados da propia base fornecida pela Data Science Brigade e pelo 
manual que pode ser acessado pela url:
https://portalnovo.tce.rs.gov.br/repo/cex/licitacon/eValidador_LicitaCon_Manual_Leiaute_1.4.pdf;

-O arquivo "rg_base.ipynb" é responsavel pela analise e resolução do problema proposto e o arquivo
"info_apoio.ipynb" pelo carregamendo da função de leitura dos dados;

-Todos os dados analisados se encontram na pasta "base";

-Todas as bibliotecas ultilizadas podem ser instaladas apartir do arquivo "requirements.txt"
pelo comando "pip install -r requirements.txt".
